# Single Source of Truth: Enforcement Protocol

**The LaTeX `.tex` file is the authoritative source for ALL paper content.** Everything else is derived.

## The SSOT Chain

```
doc/tex/main.tex (SOURCE OF TRUTH)
  ├── doc/tex/*.pdf (figures generated by scr/ scripts)
  ├── doc/tex/references.bib (bibliography)
  └── scr/ (generate figures)

NEVER edit derived artifacts independently.
ALWAYS propagate changes from source -> derived.
```

---

## Content Fidelity Rules

1. **Paper text lives in `.tex` only** -- do not duplicate paper content in scripts or markdown
2. **Figure parameters live in scripts** -- do not hardcode figure settings in `.tex`
3. **Bibliography is centralized** -- `doc/tex/references.bib`

---

## Cross-Document Consistency

When updating ANY infrastructure file, check and update ALL related documents:

| If you change... | Also update... |
|-----------------|----------------|
| A skill in `.claude/skills/` | Skills table in `CLAUDE.md` |
| A rule in `.claude/rules/` | Any reference in `CLAUDE.md` or guides |
| A template in `templates/` | Any rule that references it |
| `doc/tex/references.bib` | Verify no broken citations in papers |
| A figure script | Re-run to regenerate the figure |

---

## Figure Freshness Protocol

Before including any figure in a paper:

1. Check that the script that generates it is up-to-date
2. Re-run the script if source data or parameters have changed
3. Verify the generated figure matches expectations
4. Only then `\includegraphics` in the `.tex` file
